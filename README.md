# SpeechJammer

A system that uses daf DAF(Delayed Auditory Feedback) to jam speech.

## Requirements

- macOS
- Python ~> 3.7

## Installation

```sh
$ git clone <this repo>
$ cd <this repo>

$ pip install -r requirements.txt
```
